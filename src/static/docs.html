Mergington High School API Documentation

This API allows students to view, sign up for, and unregister from extracurricular activities at Mergington High School.

Source code: app.py | static/app.js | test_app.py

Endpoints

GET /

Redirects to the static front-end page.

Source: app.py: root()
Example:
curl -i http://localhost:8000/

GET /activities

Returns a dictionary of all activities and their details.

Source: app.py: get_activities()
Example:
curl http://localhost:8000/activities

POST /activities/{activity_name}/signup?email=EMAIL

Sign up a student for an activity.

  • Path parameter: activity_name (string)
  • Query parameter: email (string)
Source: app.py: signup_for_activity()
Example:
curl -X POST "http://localhost:8000/activities/Chess%20Club/signup?email=alice@mergington.edu"

POST /activities/{activity_name}/unregister?email=EMAIL

Unregister a student from an activity.

  • Path parameter: activity_name (string)
  • Query parameter: email (string)
Source: app.py: unregister_from_activity()
Example:
curl -X POST "http://localhost:8000/activities/Chess%20Club/unregister?email=alice@mergington.edu"

Front-End

The front-end is served at /static/index.html and uses static/app.js to interact with the API.

Testing

Unit tests are provided in test_app.py.
Run tests with: 

pytest src/test_app.py
